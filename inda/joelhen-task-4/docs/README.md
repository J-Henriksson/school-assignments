#### Exercise 1.1
| Boolean expression | Value   |
|--------------------|---------|
| `2 == 5`           | `false` |
| `2 > 5`            | `false` |
| `11 == 11`         | `true`  |
| `7 <= 9`           | `true`  |
| `42 >= 7`          | `true`  |
| `false == true`    | `false` |
